# Design reference

`silly-chat-design.html` is the canonical visual design document (generated with
Claude Design, 2026-07-10). Open it in a browser — it's a standalone gallery of
21 frames (1a–1u) covering every screen and state in both a light theme
(Frigg · Glow) and a dark mixed theme (Bifröst · Aurora).

The app should match it 100%, with these agreed deviations:
- **No serif headings** — the mock uses Georgia for display text; the app uses the
  user-selected sans (Settings → Font) everywhere, headings are just heavier weights.
- Animated backgrounds (Aurora/Mesh/Starfield) follow the app's existing CSS
  implementations; the mock's are static approximations.
