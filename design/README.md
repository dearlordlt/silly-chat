# Design reference

`silly-chat-design.html` is the canonical visual design document (generated with
Claude Design, 2026-07-10, v2). Open it in a browser — it's a standalone gallery of
26 frames (1a–1z) covering every screen and state in both a light theme
(Frigg · Glow) and a dark mixed theme (Bifröst · Aurora).

The app should match it 100%, with these agreed deviations:
- Animated backgrounds (Aurora/Mesh/Starfield) follow the app's existing CSS
  implementations; the mock's are static approximations.
