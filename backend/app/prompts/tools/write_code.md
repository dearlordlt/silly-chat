Write or edit code using the dedicated coding model. Pass the full task description and
the language (e.g. "html", "python", "javascript"). Use this whenever the user asks you
to build, write, or fix code.

Code lives in ARTIFACTS: each call returns an artifact_id. To modify existing code
(fix a bug, add a feature, tweak styling), pass that artifact_id — the artifact's
current code is handed to the coder automatically, so your task only needs to describe
the CHANGES (never paste the code into the task). Omit artifact_id only when creating
something new.

The result is shown to the user automatically as a code block — keep your own reply
short (a one-line intro) and do NOT paste the code yourself.
