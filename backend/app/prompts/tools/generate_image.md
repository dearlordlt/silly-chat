Create a brand-new image from a text description — generated by an AI image model,
not found on the web. Use it when the user asks you to draw, generate, imagine, or
design a picture ("draw me…", "make an image of…", a logo, illustration, wallpaper,
scene, character).

Write prompt as one vivid, specific description of the desired image: subject, style
(photo / watercolor / pixel art / 3D render…), setting, mood, colors. English prompts
work best. Optionally set aspect_ratio to a common ratio — "1:1" (default), "16:9"
wide, "9:16" tall, "4:3", "3:4".

Two models sit behind this tool — pick with `quality`:
- quality=false (default): the FAST model, back in seconds. Right for casual, fun,
  simple asks — memes, doodles, quick illustrations, "show me a cat in a hat".
- quality=true: the slow TOP model. Choose it when the ask demands craftsmanship —
  photorealism, rich fine detail, historical/period accuracy, complex composition,
  readable text inside the image — or the user signals stakes ("stunning",
  "super realistic", "make it perfect", a logo they'll actually use), or a fast
  attempt disappointed and they want it better.

For several distinct subjects (e.g. one portrait per party member), call
generate_image once PER subject — every call's image joins one shared gallery in
your answer. Up to 4 images per turn; if the user asks for more, generate the first
4 and say the rest need another message.

The generated images are attached to your answer automatically: never embed links,
markdown images, or your own gallery for them, and keep your reply to a short intro.
For finding real, existing photos use find_images instead.
