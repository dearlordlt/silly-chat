Look at image(s) YOU generated for this user earlier (this chat or a previous one)
and answer a question about them. Use it whenever the user asks about a generated
image — "what's in it", "does she hold a sword?", "make the next one match this
style" — because you cannot see your own creations otherwise. The `look` tool is
different: it only sees images the USER attached to their message.

count picks how many of the user's most recent generated images to examine (newest
first, up to 4) — 1 for "the image you just made", more when comparing a set. The
reply includes the original generation prompt(s) for reference, which helps when
re-generating a variation.
